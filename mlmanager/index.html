ML Manager
by Javier Santos V.
Free


ML Manager is a modern, easy and customizable APK extractor & app manager for Android that allows you to extract any installed & system app, mark them as favorite, share .apk files easily and much more.
  • Extract installed & system apps and save them as APK.
  • Batch mode to extract multiple APKs.
  • Organize your apps marking them as favorite.
  • Share APK with friends using Dropbox, Telegram, email, etc.
  • Uninstall apps easily.
  • Customizations available in Settings.
  • Material Design.
